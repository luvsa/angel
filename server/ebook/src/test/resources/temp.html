第216章天真无邪

“母妃，李功效是李贵妃的主治太医，有了他，李贵妃宫中的动静我们一概全知不说。”

“甚至还能借由李功效的手，拉拢十七皇叔。”

朱允炆将自己的想法告诉了吕氏，要让吕氏放过李功效这个人，朱允炆只能实话实说。

只有为了自己的未来，吕氏才可能就此罢休，放过李功效的狗命。

谁知道，朱允炆的话音刚落，吕氏却哈哈大笑了起来。

“哈哈哈哈，我的儿呀，你可真是天真的很呀，哈哈哈……”

吕氏笑了半天才停下来，看着朱允炆惨绿的脸色，吕氏才收起了自己的笑声。

“我的儿，你竟然会有拉拢朱权的想法，真不知道母后应该夸你自信，还是应该说你天真。”

“母妃这话什么意思！”

朱允炆脸色铁青。

自己母妃竟然这般不顾及形象的嘲笑自己，朱允炆的脸上有些挂不住。

“允炆，你就没想过，你皇爷爷为什么将所有成年的皇子发配到了籓地，唯独留下了老十七吗。”

“甚至，连你皇爷爷最重视的朵颜三卫，都留给了老十七。”

“你就没有用你那聪明的小脑瓜，仔细想想吗？”

“妄图拉拢朱权，你还真是可笑呀。”

吕氏虽然面上笑着，可是却真是恨不得掐死自己眼前这个亲生的儿子。

她一直以为，她对朱允炆的教导，也算是教育有方。

可是这孩子，一点都没有遗传到她的聪明才智，只遗传了她的阴狠毒辣。

也不知道她到底是该笑还是该哭。

“母妃，你什么意思，皇爷爷，不可能传位给十七皇叔的。”

朱允炆想不明白。

在他的心目中，哪怕朱元璋传位给四皇叔燕王朱棣，都比传位给朱权的可能性大。

自己那个十七皇叔，生性贪玩，无心朝政，更是没少惹得朱元璋生气。

所以，再怎么轮，朱元璋都不会把皇位交到朱权的手中。

“母妃说你天真，你还真就开始无邪了。”

“你以为你十七皇叔真的像是他表现的那般无能吗。”

“连你父亲都对他称赞有加，赞叹不止。”

“朱权真的就那么烂泥扶不上墙头吗？”

“他只是无心那个位置罢了，但是这不代表你皇爷爷不重视他。”

“那么的皇子，成年之后全部赴任藩地，唯独老十七一留多年。”

“你真的以为是因为一个不受宠的李贵妃吗？”

“比起朱允熥，老十七才是你最大的对手，那家伙能文能武，甚至有勇有谋。”

“带兵打仗，丝毫不逊色你四皇叔，甚至跟蓝玉都有的一拼。”

“虽然不是嫡子，但是你皇爷爷对他的重视，显然比你还高。”

“不要天真的把敌人当成朋友。”

“你最应该做的，是挑拨你皇爷爷和老十七的关系，让你皇爷爷尽早把老十七赶出皇城。”

“他们两个关系的纽带，只在一个李贵妃，只有李贵妃死了……”

“这样，你才能高枕无忧。”

吕氏说完这句话，只是冷笑了一声，便扔下了朱允炆，径直出了门去。

李功效她是不会留下的，不过，因为和朱允炆的的一番话，倒是让她有了主意。

李贵妃必须死，当然，最好是死在李功效的手中，
